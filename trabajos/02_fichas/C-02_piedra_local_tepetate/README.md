# C-02: piedra local o "tepetate"

Pendiente llenar:

- Comunidad.
- Uso actual o recordado.
- Materiales observados: piedra local, cubierta, piso, posibles juntas o muro a hueso.
- Estado físico.
- Cambios visibles o relatados.
- Relato asociado.
- Fotografías publicables e internas.
- Ubicación generalizada para tesis.

Nota: usar el nombre `tepetate` con cautela porque falta caracterización material precisa.
