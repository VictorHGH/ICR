# C-04: órganos

Pendiente llenar:

- Comunidad.
- Uso actual o recordado.
- Función: cerramiento, protección, delimitación, sombra u otra.
- Estado físico o vitalidad de los órganos.
- Relación con patio, caminos o área doméstica.
- Cambios visibles o relatados.
- Relato asociado.
- Fotografías publicables e internas.
- Ubicación generalizada para tesis.

Tratar como cerramiento vivo o semivivo, no necesariamente como edificio convencional.
