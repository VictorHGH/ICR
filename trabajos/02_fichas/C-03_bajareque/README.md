# C-03: bajareque

Pendiente llenar:

- Comunidad.
- Uso actual o recordado.
- Materiales observados: entramado de lechuguilla, relleno, recubrimiento con tierra, cubierta, piso.
- Estado físico.
- Cambios visibles o relatados.
- Relato asociado.
- Fotografías publicables e internas.
- Ubicación generalizada para tesis.

Usar siempre `bajareque` como término del sistema.
