# Relatos

Guardar aquí audios, transcripciones y síntesis por caso.

## Estructura

- `01_audios_originales/`: audios completos, si existen.
- `02_transcripciones/`: transcripciones o notas extensas.
- `03_sintesis_por_caso/`: resumen breve listo para ficha.

## Cuidado

- No incluir nombres reales en archivos que se usarán en ICR.
- Usar códigos de fuente si hace falta: `R-01`, `R-02`, etc.
- Separar información sensible de información publicable.
