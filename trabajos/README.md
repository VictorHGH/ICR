# Organización de trabajo para cierre de ICR

Esta carpeta organiza el material pendiente para cerrar el diagnóstico descriptivo y cartográfico. La regla principal es trabajar solo con evidencia existente: fotografías, geolocalizaciones, mapas, relatos y notas disponibles. No se contempla una nueva visita de campo.

## Orden de trabajo

1. Completar `01_corpus_y_datos/matriz_corpus_C01_C04.md` con los datos mínimos de cada construcción.
2. Llenar una ficha por caso en `02_fichas/` usando `02_fichas/plantilla_ficha.md`.
3. Ordenar mapas y capas en `03_cartografia/`.
4. Clasificar fotografías en `04_fotografias/` y dentro de cada ficha por caso.
5. Ordenar audios, transcripciones y síntesis en `05_relatos/`.
6. Completar cuadros comparativos en `06_tablas_y_sintesis/`.
7. Copiar únicamente las imágenes listas para tesis en `07_figuras_para_tesis/`.
8. Revisar pendientes en `08_pendientes_y_revision/pendientes.md`.

## Casos actuales

- `C-01`: penca de maguey.
- `C-02`: piedra local o "tepetate"; mantener cautela porque falta caracterización material.
- `C-03`: bajareque.
- `C-04`: órganos como cerramiento vivo o semivivo.

## Criterio de privacidad

- Usar coordenadas exactas solo para análisis interno.
- En la tesis, usar ubicación generalizada.
- No publicar nombres de propietarios ni fotografías sensibles sin autorización.
- Separar fotografías publicables de fotografías internas.
