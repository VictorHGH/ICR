\section{Metodología}

Este capítulo detalla el diseño cualitativo orientado a construir un diagnóstico descriptivo y cartográfico de cuatro casos seleccionados de construcciones tradicionales hñähñu. La selección parte de un registro más amplio de construcciones observadas, pero se concentra en los casos que representan con mayor claridad sistemas constructivos originales por su morfología y materiales tradicionales. La metodología se organiza de acuerdo con una relación explícita entre unidad de análisis, categorías, preguntas, técnicas y productos, siguiendo la lógica de acotar la investigación a evidencias verificables \citep{guerrerobaca2025propuesta}. El marco teórico-conceptual aporta categorías para guiar la lectura, pero el trabajo empírico se concentra en materiales disponibles: fotografías, georreferenciación, recorridos, relatos hablados y notas de observación.

%--------------------------------------------------------------------
\subsection{Diseño general de investigación}
%--------------------------------------------------------------------

\begin{itemize}
	\item \textbf{Enfoque:} estudio de caso múltiple con lógica etnográfica
	      y descriptiva.
	\item \textbf{Ámbito espacial:} comunidades ``El Deca'' y ``El Buena'',
	      municipio de El Cardonal, Hidalgo.
	\item \textbf{Horizonte temporal:} transformaciones de largo plazo
	      reconstruidas por relatos (aprox.
	      últimas décadas) y comparación fotográfica en dos cortes
	      (registro previo vs. registro actual) cuando sea posible.
	\item \textbf{Muestra:} \emph{n} = 4 casos seleccionados de construcciones tradicionales
	      (principalmente cocinas, bodegas, resguardos o cerramientos), elegidos por muestreo
	      intencional dentro del registro disponible para representar diversidad de sistemas constructivos,
	      morfologías y materiales tradicionales.
    \item \textbf{Unidad de análisis:} cada construcción documentada como caso independiente, identificada mediante clave anónima.
\end{itemize}

\subsection{Estrategias de recolección de datos}
%--------------------------------------------------------------------

\vspace{-0.5em}
\begin{enumerate}
	\item \textbf{Inventario fotográfico y fichas descriptivas}\\
	      \textit{Instrumentos:} registro fotográfico sistemático, notas de campo y fichas de observación.\\
	      \textit{Producto:} fichas de construcciones y evidencias por sistema constructivo.
	\item \textbf{Georreferenciación y cartografía}\\
	      \textit{Instrumentos:} puntos y recorridos (archivos \texttt{shape}) trabajados en QGIS.\\
	      \textit{Producto:} mapas de contexto (México--región--comunidades) y mapas a escala local con localización de construcciones.
	\item \textbf{Relatos hablados y entrevistas abiertas}\\
	      \textit{Sujetos:} habitantes/propietarios y, cuando sea posible, portadores del saber constructivo. \\
	      \textit{Eje temático:} experiencias de habitar, decisiones de mantenimiento o sustitución, acceso a materiales y cambios observados.
	\item \textbf{Revisión documental focalizada}\\
	      Lectura de bibliografía clave para respaldar la descripción de sistemas y sus transformaciones.
	\item \textbf{Comparación temporal (antes/después) cuando sea posible}\\
	      Contraste de fotografías previas con un registro actual para evidenciar cambios y permanencias.
\end{enumerate}

%--------------------------------------------------------------------
\subsection{Procedimiento de trabajo}
%--------------------------------------------------------------------

El procedimiento se organiza en ocho pasos sucesivos:

\begin{enumerate}
  \item Ordenar la matriz de corpus C-01 a C-04 con comunidad, sistema preliminar, uso, materiales, estado físico, relato, fotografías y ubicación generalizada, aclarando que se trata de casos seleccionados dentro de un registro mayor.
  \item Separar fotografías publicables e internas, identificando las imágenes útiles para cada ficha.
  \item Integrar para cada caso una ficha descriptiva con los campos mínimos definidos en el producto operativo.
  \item Revisar audios, notas o transcripciones para elaborar una síntesis breve del relato asociado.
  \item Organizar puntos, recorridos y capas cartográficas en QGIS, diferenciando información interna y versión generalizada para la ICR.
  \item Comparar los cuatro casos por sistema constructivo, materiales, uso, estado y cambios visibles o relatados.
  \item Elaborar mapas y cuadros de síntesis que permitan leer relaciones entre materialidad, territorio y memoria.
  \item Formular criterios básicos de documentación y conservación preventiva derivados del diagnóstico.
\end{enumerate}

%--------------------------------------------------------------------
\subsection{Técnicas de análisis}
%--------------------------------------------------------------------

\begin{itemize}
	\item \textbf{Tipología de sistemas constructivos} a partir de rasgos observables en el registro fotográfico
	      (materiales, ensambles, cubiertas, acabados) y de los relatos de los habitantes.
	\item \textbf{Análisis de contenido} de relatos hablados para extraer categorías recurrentes
	      (materiales disponibles, proximidad, mantenimiento, sustitución, sentido de uso).
	\item \textbf{Análisis cartográfico} para describir la localización de construcciones y recorridos, así como su relación
	      con el contexto territorial y la disponibilidad de recursos.
	\item \textbf{Triangulación} entre fotografías, cartografía y relatos para sustentar los hallazgos y construir
	      láminas de diagnóstico (mapas + fichas + evidencia fotográfica).
\end{itemize}

\subsection{Consideraciones éticas y de devolución}
%--------------------------------------------------------------------

\begin{enumerate}
	\item Consentimiento informado bilingüe (español/hñähñu).
	\item Anonimato codificado de entrevistados.
	\item Manejo diferenciado de localización: para presentaciones internas se podrá utilizar la ubicación exacta de las construcciones; para la versión pública se generalizarán coordenadas y se anonimizarán claves y fotografías cuando sea necesario.
	\item Devolución de resultados mediante fichas sintéticas, mapas generalizados y, si es necesario, un cuadernillo ilustrado derivado del diagnóstico.
\end{enumerate}

%--------------------------------------------------------------------
\subsection{Limitaciones del estudio}
%--------------------------------------------------------------------

\begin{itemize}
	\item \textbf{Representatividad geográfica} limitada a dos comunidades;
	      transferibilidad se discutirá comparando literatura de casos
	      análogos (Mixteca, Sierra Norte de Puebla).
	\item \textbf{Tiempo de campo} restringido a la posibilidad temporal del investigador;
	      algunos rituales estacionales podrían no ser observados en directo.
	\item \textbf{Ausencia de levantamientos arquitectónicos}: el estudio se limita a un registro fotográfico
	      y descriptivo, sin mediciones detalladas de plantas, secciones o componentes.
	\item \textbf{Alcance testimonial}: los relatos se emplean como memoria y experiencia asociada a cada caso, no como muestra representativa de opinión comunitaria.
    \item \textbf{Caracterización material preliminar}: algunos nombres locales, como ``tepetate'', se mantendrán con cautela mientras no exista caracterización técnica del material.
\end{itemize}

%--------------------------------------------------------------------
\subsection{Productos esperados}
%--------------------------------------------------------------------

Con esta metodología se busca producir un diagnóstico verificable y útil para la discusión: matriz de corpus, fichas descriptivas, tipología inicial de sistemas constructivos, mapas de contexto, síntesis comparativa y criterios básicos de documentación y conservación preventiva.
